# 家庭账本平台实现计划

## 技术栈
- **后端**: FastAPI + SQLAlchemy + SQLite
- **前端**: Vue.js 3 + Element Plus + ECharts
- **数据库**: SQLite (开发和小规模使用)

## 核心功能模块
1. **账户管理** - 多种账户类型支持(现金、银行卡、支付宝、微信等)
2. **收支流水录入** - 收入/支出/转账记录,支持快速录入
3. **分类体系** - 多级分类,支持自定义
4. **统计分析** - 月度汇总、分类占比、趋势分析
5. **预算管理** - 月度预算设置和执行跟踪
6. **报表导出** - Excel/CSV导出

## 项目结构
```
family-finance/
├── backend/          # FastAPI后端
├── frontend/         # Vue.js前端
├── data/            # SQLite数据库
└── requirements.txt
```

## 核心数据表
- accounts (账户)
- categories (分类)
- transactions (流水)
- budgets (预算)
